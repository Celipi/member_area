Guia de Variáveis de Template

Use variáveis de template para personalizar suas mensagens para cada aluno. Quando a mensagem for enviada, estas variáveis serão substituídas automaticamente pelos dados reais.

Dica: Mensagens personalizadas têm taxas de abertura e engajamento muito maiores. Use o nome do aluno e detalhes específicos do curso para criar uma experiência mais personalizada.

Variáveis Disponíveis

[[name]]

Nome Completo

Exibe o nome completo do aluno.

Ex: "João Silva Pereira"

[[first_name]]

Primeiro Nome

Exibe apenas o primeiro nome do aluno. Ideal para saudações.

Ex: "João"

[[email]]

Email do Aluno

Exibe o endereço de email utilizado para acesso à plataforma.

Ex: "joao.silva@exemplo.com"

[[password]]

Senha de Acesso

Exibe a senha temporária gerada para o primeiro acesso do aluno.

Ex: "Abc123XyZ"

[[curso]]

Nome do Curso

Exibe o nome do curso ao qual o aluno recebeu acesso.

Ex: "Curso Avançado de Marketing Digital"

[[link]]

Link de Acesso Padrão

Exibe o link para acesso à plataforma, onde o aluno precisará inserir email e senha.

Ex: "https://seudominio.com/login"

[[fast_link]]

Link de Acesso Rápido

Exibe um link especial que leva o aluno diretamente para a área de membros, sem necessidade de login.

Ex: "https://seudominio.com/acesso-rapido/xyz123abc"
